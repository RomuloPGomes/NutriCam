# NutriCam PWA para iPhone

Esta pasta esta pronta para publicar como site HTTPS e usar no iPhone via Safari.

## Publicar no Netlify

1. Acesse https://app.netlify.com/drop
2. Arraste a pasta `NutriCam_PWA_iOS` para a pagina.
3. Aguarde o Netlify gerar um link HTTPS.
4. Abra o link no Safari do iPhone.
5. Toque em Compartilhar.
6. Toque em Adicionar a Tela de Inicio.
7. Abra o NutriCam pelo icone criado.

## Publicar no Vercel

1. Crie um projeto estatico.
2. Envie todos os arquivos desta pasta.
3. Use a raiz da pasta como diretorio publico.

## Observacoes iOS

- A camera funciona pelo botao Tirar foto e analisar.
- A IA precisa de internet e da chave Gemini salva no Perfil.
- O iOS exige HTTPS para instalar como PWA.
- Notificacoes no iPhone dependem da versao do iOS e de permissao do usuario.
